Fixture provenance — external leg (Illumina STR FASTQ)
======================================================

sample.fastq is REAL data: the first 5,000 reads of a single-source sample from
the NIST Forensic DNA Open Dataset (mds2-2157), Verogen ForenSeq DNA Signature
Prep Kit (Illumina MiSeq). Same donor (NTD01) as the PowerSeq fixture in
tools/strait-razor-PowerSeqv2.31/fixtures/nist-mds2-2157/, so the two assay
paths are directly comparable.

NIST mds2-2157 covers multiple Illumina STR kits (ForenSeq, PowerSeq 46GY, …).
STRhub Verified has published examples for both ForenSeq and PowerSeq. This
directory holds the ForenSeq slice used for the typed external leg; tool
manifests should point their own fixture to kit-matched reads (see
fixtures/nist-mds2-2157/README.md).

Dataset : NIST mds2-2157  (DOI 10.18434/M32157)
Landing  : https://data.nist.gov/od/id/mds2-2157
Use scope: research / training / education only; not for donor identification or
           database searching (per NIST). A smoke test is within scope.

Source file inside the dataset zip:
  NGS_Assays/Verogen_ForenSeq_DNA_Signature_Prep_Kit.zip
  -> Verogen_ForenSeq_DNA_Signature_Prep_Kit/Signature_Prep_DPMB_Single_Source/FS_NTD01.fastq
  (donor NTD01; ForenSeq single-source files are combined, not split R1/R2)

Matching STRait Razor config: ForenSeqv1.27.config  (used in manifest.yml)

How it was built (no multi-GB download; HTTP range reads into the zip):
  python -m pip install remotezip
  python - <<'PY'
  import io
  from remotezip import RemoteZip
  url="https://data.nist.gov/od/ds/ark:/88434/mds2-2157/NGS_Assays/Verogen_ForenSeq_DNA_Signature_Prep_Kit.zip"
  name="Verogen_ForenSeq_DNA_Signature_Prep_Kit/Signature_Prep_DPMB_Single_Source/FS_NTD01.fastq"
  with RemoteZip(url) as z, z.open(name) as fh, open("sample.fastq","wb") as w:
      r=io.BufferedReader(fh)
      for _ in range(5000):
          rec=[r.readline() for _ in range(4)]
          if not rec[3]: break
          w.writelines(rec)
  PY

seqkit stats: 5,000 reads, fixed 351 bp.
